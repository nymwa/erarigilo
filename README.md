# ERARIGILO

error generation tool

## usage (en)

note: please use SpaCy v3. SpaCy v2 is not supported.

- `erg ready en`
	- SpaCy Tokenization & Tagging
- `erg run en`
	- Error Generation
	- Please write which error modules you use in `config.yaml`.
- `erg show en`
	- Show Generated Errors
- `erg form en`
	- Make parallel data
